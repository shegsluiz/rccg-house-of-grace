## 2024-05-09 - Accessible interactive elements and external links
**Learning:** Found that icon-only buttons (like the mobile menu toggle) and social links in the footer lacked proper `aria-label`s, focus states for keyboard navigation, and secure `rel` attributes for external links. Also learned to not commit `pnpm-lock.yaml` unless explicitly required when it's generated by installing dependencies.
**Action:** Always add descriptive `aria-label` and `focus-visible:ring-2 focus-visible:ring-hog-green-400 focus:outline-none` styles to icon-only buttons/links, and ensure external links (`target="_blank"`) use `rel="noopener noreferrer"`. Ensure `pnpm-lock.yaml` isn't accidentally committed when keeping PR size small.
